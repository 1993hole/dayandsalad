# assets

고객 제공 이미지가 들어갈 자리입니다.

- `logo/`   — 상단 로고 (예: logo.png) → index.html 헤더의 `<h1>` 교체
- `banner/` — 홈 상단 컴백 배너 슬라이드 이미지 (예: banner1.jpg ~ banner3.jpg)
- `menu/`   — 메뉴 5종 사진 (예: menu1.jpg ~ menu5.jpg) → js/menu-data.js 의 grad 대신 사용
- `track/`  — 배달추적 화면 지도 자리에 들어갈 연출 이미지 (예: map.jpg)

이미지를 받으면 위 폴더에 넣고, 코드의 placeholder(그라데이션·이모지)를 해당 경로로 교체합니다.
