# assets — 에셋 파일명 가이드

기업 제공 에셋이 들어갈 자리입니다. **아래 파일명 그대로** 각 폴더에 넣으면 코드가 인식합니다.
(에셋 도착 후 `js/menu-data.js`·`index.html`의 placeholder를 해당 경로로 연결)

## 폴더 / 파일명

```
assets/
├─ logo/
│   └─ logo.svg          # 로고 — 1순위 벡터(SVG). 없으면 logo.png (고해상도·투명배경)
│                        #   권장: 높이 60px 기준, 가로 비율 자유
├─ banner/
│   ├─ banner1.jpg       # 홈 상단 슬라이드 1
│   ├─ banner2.jpg       # 홈 상단 슬라이드 2
│   └─ banner3.jpg       # 홈 상단 슬라이드 3        (각 768×360, 약 2:1)
├─ member/
│   ├─ member1.jpg       # 현빈 · 오로라 그린 볼
│   ├─ member2.jpg       # 윤 · 미드나잇 프로틴 볼
│   ├─ member3.jpg       # 연우 · 베리 하모니 볼
│   ├─ member4.jpg       # 진혁 · 키토 아보카도 볼
│   └─ member5.jpg       # 시윤 · 앙코르 시그니처 볼
│                        #   멤버당 1장이면 충분 — 홈 라인업 / 메뉴 썸네일 /
│                        #   상세 상단 / 배달현황(공개 후)에 공통 사용(자동 크롭)
│                        #   권장: 큰 세로형 1080×1350
├─ track/
│   ├─ blind.jpg         # 배달현황 '컨셉 공개 전' 블라인드 (840×480, 공통 1장)
│   │                    #   메뉴별로 다르게 하려면 menu-data.js 의 TRACK_BLIND_IMG 수정
│   │  ── 아래는 '컨셉 공개(7/20 18:00)' 후 멤버 컨셉포토. 공개 직전(17:55쯤) 업로드 ──
│   ├─ reveal1.jpg       # 현빈 (오로라 그린 볼)   840×480
│   ├─ reveal2.jpg       # 윤 (미드나잇 프로틴 볼)
│   ├─ reveal3.jpg       # 연우 (베리 하모니 볼)
│   ├─ reveal4.jpg       # 진혁 (키토 아보카도 볼)
│   └─ reveal5.jpg       # 시윤 (앙코르 시그니처 볼)
│                        #   ※ 업로드 전엔 파일이 없어 블라인드 유지. 정각(18:00) 업로드 시 노출창 0
├─ ad/
│   └─ ad.jpg            # 메뉴페이지 광고 배너 (768×192, 선택)
└─ icons/                # ※ 아이콘: 현재 이모지 사용 중. 커스텀 SVG로 교체 예정(기업 협의 후 확정)
    ├─ home.svg          #   (예정) 홈 탭
    ├─ delivery.svg      #   (예정) 배달현황 탭
    └─ share.svg         #   (예정) 공유 버튼
```

## 크기·형식 요약

| 에셋 | 권장 크기(2x) | 형식 |
| --- | --- | --- |
| 로고 | 높이 60px 기준 | **SVG**(우선) / PNG(투명) |
| 배너 슬라이드 | 768 × 360 | JPG |
| 멤버 사진 | 1080 × 1350 (세로형) | JPG |
| 배달현황 블라인드 | 840 × 480 | JPG |
| 광고 배너 | 768 × 192 | JPG/PNG |
| 아이콘 | 벡터 | **SVG** (예정) |

## 참고
- **사진(멤버·배너·블라인드)은 SVG 불가** — 래스터라 JPG 유지. SVG는 로고·아이콘·단순 그래픽에만.
- 멤버 사진을 자리마다 다른 컷으로 쓰려면 `member1-hero.jpg`처럼 추가 가능(선택).
- 현재 멤버 사진은 홈·메뉴엔 바로 노출되고, 배달현황만 공개 전 블라인드 → 전체를 공개 전 가리려면 별도 처리.
